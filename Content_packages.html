    Content packages

    BMC Helix Innovation Studio provides content packages for out-of-the-box Robotic Process Automation connectors such as Automation Anywhere, Blue Prism, and other connectors. Use these content packages to integrate with third-party applications and automate service requests. A content package includes sample data, such as services, workflows, questionnaires, definitions, and bot scripts. The sample services are available in your BMC Helix Digital Workplace Catalog application by default.

    As an application business analyst, you can leverage the content packages to implement services that automatically manage user requests. You can also recreate and customize the sample services to implement numerous services.

    The following sample BMC Helix Digital Workplace Catalog services are provided to automate the following actions:

    Service

    Description

    BMC Sample Create Virtual Machine 21.05

    Creates a virtual machine in Microsoft Azure.

    BMC Sample Request Payslip 21.05

    Fetches payslips from a payroll system by using an Automation Anywhere Robotic Process Automation (RPA) bot.

    BMC Sample Deploy Instance 21.05

    Runs a Jenkins job.

    BMC Sample Request Time Off 21.05

    Applies a time off by using a Blue Prism process.

    BMC Sample Request Payslip with Blue Prism 21.3.01

    Fetches payslips from a payroll system by using a Blue Prism Robotic Process Automation (RPA) bot.

    BMC Sample Request Payslip with UiPath 21.3.01

    Fetches payslips from a payroll system by using a UiPath process.

    BMC Sample AutomationEdge Content Pack 22.1

    Adds an employee to the system using the AutomationEdge Robotic Process Automation (RPA) bot.

    BMC Sample AutomationEdge Content Pack With Callback 22.1

    Creates an employee using the AutomationEdge Robotic Process Automation (RPA) bot, with the callback function.

    These services are included in your development environment. You can customize the content and test it in your QA environment before you deploy it into production. BMC Helix Innovation Studio provides a utility to export and import your custom content between environments. Learn about exporting and importing content packages at Deploying application tailoring changes .
